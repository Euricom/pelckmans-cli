# Usage

<!-- usage -->
```sh-session
$ npm install -g @euricom/pelckmans-cli
$ pelckmans-cli COMMAND
running command...
$ pelckmans-cli (-v|--version|version)
@euricom/pelckmans-cli/0.0.4 darwin-x64 node-v14.17.0
$ pelckmans-cli --help [COMMAND]
USAGE
  $ pelckmans-cli COMMAND
...
```
<!-- usagestop -->

```sh-session
npm i
npm run start

OR (if you have yarn)
yarn
yarn start
```
